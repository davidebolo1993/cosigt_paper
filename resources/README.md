# Resources

Sample lists, assembly identifiers, benchmarking regions, and prediction results used in the manuscript.

## Structure

- **[alignments/](alignments)** - 1000 Genomes sample lists
- **[assemblies/](assemblies)** - Assembly lists from HGSVC and HPRC pangenomes
- **[benchmarking/](benchmarking)** - COSIGT (and Locityper, where available) prediction results on benchmarking regions
- **[reference/](reference)** - GRCh38 reference genome

Each subdirectory contains a dedicated `README.md` with more detailed information.
